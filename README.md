# PHP CRUD Operation

Simple PHP CRUD (Create, Read, Update, Delete) project using:
- PHP
- MySQL
- HTML

## Features
- Add student record
- View records
- Edit record
- Delete record

## How to Run
1. Install XAMPP
2. Copy project to htdocs folder
3. Create database and table
4. Run in browser: http://localhost/php-crud-operation/
